Lab Report -02

Problem Statement

In this task, you have to create a photo gallery of the faculty profile of the Green University of Bangladesh using HTML and CSS. You have to show a total of 12 faculty images to create this photo gallery. In this experiment, it will show six images side by side

  1. For showing the description of each faculty you have to use the font size 15px, text- align=left.

  2. You have to use padding/grid-gap to show the space between the two images

  3. You have to show the name, designation, and email address of each faculty in the description field

  4. To show the "View Profile" button.

  5. The faculty images should be clickable which means when we click an image of a faculty the image should be focused and should be shown a bit larger.

  6. You can use any of the 12 GUB faculty's images to create this photo gallery.
